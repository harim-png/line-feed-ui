# LINE Feed UI

Figma 디자인을 기반으로 구현한 LINE 소셜 피드 UI입니다.

## 설치

```bash
npm install
```

## 실행

```bash
npm run dev
```

브라우저에서 http://localhost:5173 을 열어서 확인하세요.

## 기술 스택

- React 18
- Vite
- Tailwind CSS

## 디자인 출처

Figma: https://www.figma.com/design/5dq1rFAfclGMZsma9Y383U/Cursor-Test?node-id=0-3057

## 주의사항

이미지 에셋들은 Figma API에서 제공하는 임시 URL을 사용합니다. 7일 후 만료될 수 있으니, 실제 프로덕션에서는 로컬 이미지로 교체해야 합니다.
